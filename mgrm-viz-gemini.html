Interactive Graded Response Model (GRM) Visualization

How to Use This Tool:

Adjust the sliders to see how Theta (θ), Discrimination (a), the Number of Cutpoints, and individual Cutpoint (bk) values affect category response probabilities and the item‑information curve.

  • Theta (θ): Person's trait level.
  • Discrimination (a): Item's ability to differentiate.
  • Number of Cutpoints: Determines the number of response categories (Categories = Cutpoints + 1).
  • Cutpoints (bk): Thresholds on the theta scale (b1 ≤ b2 ≤ …).
