Interactive Graded Response Model (GRM) Visualization

How to Use This Tool:

This tool visualizes the Graded Response Model (GRM). Adjust the sliders to see how Theta (θ), Discrimination (a), the Number of Cutpoints, and individual Cutpoint (bk) values affect category response probabilities.

  • Theta (θ): Person's trait level.
  • Discrimination (a): Item's ability to differentiate.
  • Number of Cutpoints: Determines the number of response categories (Categories = Cutpoints + 1).
  • Cutpoints (bk): Thresholds on the theta scale. Must be ordered (b1 ≤ b2 ≤ ...).
