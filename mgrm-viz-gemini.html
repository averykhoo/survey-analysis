Interactive Graded Response Model (GRM) Visualization

How to Use This Tool:

This tool visualizes the Graded Response Model (GRM), a common model in Item Response Theory (IRT). It shows how the probability of selecting different ordered response categories for a question changes based on a person's underlying trait level (Theta) and the characteristics of the question itself (Discrimination 'a' and Cutpoints 'b').

  • Theta (θ): Represents the person's trait level (e.g., ability, attitude). Higher theta means more of the trait.
  • Discrimination (a): Indicates how well the item differentiates between people with different trait levels. Higher 'a' means steeper probability curves.
  • Cutpoints (b1, b2, b3): These are thresholds on the theta scale. For an item with 4 response categories (0, 1, 2, 3), there are 3 cutpoints.
    • b1: Point related to transitioning from Category 0 to Category 1 or higher.
    • b2: Point related to transitioning from Category 1 to Category 2 or higher.
    • b3: Point related to transitioning from Category 2 to Category 3.
    The cutpoints must be ordered (b1 ≤ b2 ≤ b3). The sliders will enforce this.

Note: You mentioned MGRM (Multidimensional GRM). This visualization is for a unidimensional GRM. An MGRM would involve multiple theta dimensions and 'a' vectors, which is more complex to visualize interactively in this format.
