Interactive Graded Response Model (GRM) Visualization

How to Use This Tool:

Adjust the sliders to explore how Theta (θ), Discrimination (a), the Number of Cutpoints, and each Cutpoint (bk) influence category probabilities, item information, and more.

  • Theta (θ): Person's trait level.
  • Discrimination (a): Item's ability to differentiate between trait levels.
  • Number of Cutpoints: Response categories = cutpoints + 1.
  • Cutpoints (bk): Ordered thresholds along θ (b1 ≤ b2 ≤ …).
